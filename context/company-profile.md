## Product Overview

### TechCorp Platform

TechCorp Platform is a **cloud-based project management SaaS** designed for modern teams. It combines task management, team collaboration, time tracking, and reporting into a single unified platform accessible from any browser or mobile device.

**Key Capabilities:**
- Project and task management with customizable workflows
- Real-time team collaboration and messaging
- Time tracking and resource allocation
- Dashboards and reporting with export capabilities
- Integrations with popular development and communication tools
- REST API for custom integrations and automation
- Role-based access control and enterprise security features

**Supported Platforms:**
- Web application (Chrome, Firefox, Safari, Edge)
- iOS mobile app (v14.0+)
- Android mobile app (v10.0+)
- Desktop app (Windows, macOS, Linux via Electron)

## Pricing

### Pricing Tiers

All plans are billed per team. A "team" is defined as a workspace with shared projects and members.

| Tier             | Price              | Included Members | Storage     | Key Features                              |
|------------------|--------------------|------------------|-------------|-------------------------------------------|
| **Starter**      | $29/month per team | Up to 10 members | 5 GB        | Basic projects, tasks, Kanban boards, email support |
| **Professional** | $79/month per team | Up to 50 members | 50 GB       | All Starter features + time tracking, reporting dashboards, integrations (Slack, GitHub), priority support |
| **Enterprise**   | Custom pricing     | Unlimited        | Unlimited   | All Professional features + SSO/SAML, audit logs, dedicated account manager, custom integrations, SLA guarantees, API access |
